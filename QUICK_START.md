# 快速开始指南

## 🚀 5分钟上手和声游戏

### 第一步：确认环境

确保你已经安装了 Node.js：

```cmd
node --version
```

应该显示 v18 或更高版本。

### 第二步：安装依赖

在项目目录下运行：

```cmd
npm install
```

这将安装所有必需的依赖包（约340个包）。

### 第三步：启动应用

```cmd
npm run dev
```

应用将自动在浏览器中打开，地址为: http://localhost:5173/

### 第四步：开始学习

1. **选择关卡**: 点击第一个关卡（蓝色）
2. **查看练习题**: 阅读练习题说明
3. **编辑和弦**: 点击和弦，编辑四个声部的音符
4. **提交验证**: 点击"提交"按钮
5. **查看反馈**: 如果有错误，查看错误提示并修正
6. **完成练习**: 正确后自动解锁下一关

## 📖 界面说明

### 关卡选择界面
- 🟢 **绿色**: 已完成
- 🔵 **蓝色**: 可以开始
- ⚫ **灰色**: 尚未解锁
- 💫 **脉动**: 当前关卡

### 练习题界面
- **五线谱**: 显示当前的和弦进行
- **和弦编辑器**: 点击和弦进行编辑
- **音符输入**: 选择音高、变音记号、八度
- **错误显示**: 显示验证错误（如果有）
- **操作按钮**: 提交、显示答案、跳过

## 🎵 基本操作

### 添加和弦
1. 点击和弦编辑器中的 "+" 按钮
2. 编辑四个声部的音符
3. 点击"确认"

### 编辑和弦
1. 点击要编辑的和弦
2. 修改任意声部的音符
3. 点击"确认"

### 删除和弦
1. 点击和弦上的 "×" 按钮

### 提交答案
1. 完成和弦进行编辑
2. 点击"提交"按钮
3. 查看验证结果

### 查看答案
1. 点击"显示答案"按钮
2. 参考答案将显示在五线谱上

## ⚠️ 常见错误

### 平行五度
两个声部同时以完全五度的音程移动。

**解决方法**: 改变其中一个声部的音符，避免平行移动。

### 平行八度
两个声部同时以八度的音程移动。

**解决方法**: 改变其中一个声部的音符，避免平行移动。

### 声部交叉
相邻声部的音高顺序错误（如女低音高于女高音）。

**解决方法**: 调整音符的八度，确保声部顺序正确。

### 声部超出范围
音符超出了声部的合理音域。

**解决方法**: 选择合适的八度，确保音符在声部范围内。

## 🎯 学习建议

### 初学者
1. 从第一关开始，按顺序完成
2. 仔细阅读错误提示
3. 遇到困难时查看参考答案
4. 重复练习已完成的关卡

### 进阶学习
1. 尝试不看答案完成练习题
2. 探索不同的和弦进行方式
3. 理解为什么某些进行是错误的
4. 总结和声写作的常见模式

## 🔧 故障排除

### 应用无法启动
```cmd
# 清除缓存并重新安装
rmdir /s /q node_modules
del package-lock.json
npm install
npm run dev
```

### 进度丢失
进度保存在浏览器的 LocalStorage 中。如果清除了浏览器数据，进度会丢失。

### 五线谱显示异常
刷新页面（F5）通常可以解决显示问题。
