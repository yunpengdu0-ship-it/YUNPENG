# 快速修复 - 页面空白问题

## 问题原因
`GameManager` 的接口与 `GameContext` 期望的不匹配，导致 JavaScript 错误。

## 解决方案
创建了 `GameManagerFacade` 类作为桥接层，提供 `GameContext` 需要的简化接口。

## 修复的文件
1. ✅ `src/game/GameManagerFacade.ts` - 新建门面类
2. ✅ `src/context/GameContext.tsx` - 更新导入和初始化
